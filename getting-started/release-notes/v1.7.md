# KubeArmor v1.7 Release Notes

**Date:** 2025-11-28
**Version:** v1.7.0

## Improvements

- Improved KubeArmor operator handling of image configuration, centralizing image, pull secret, toleration, node selector, and environment updates for the agent, init, relay, and controller components.

## Bug Fixes

- Fixed resource and map handling issues in the system monitor and IMA hash integration to improve compatibility and stability on kernels such as RHEL and other 5.4-based kernels.
