# KubeArmor v1.6.4 Release Notes

**Date:** 2025-11-28  
**Version:** v1.6.4

## New Features

- Added support for pulling images from private registries.

## Improvements

- Improved overall release robustness.

## Bug Fixes

- Fixed system monitor compilation failures on RHEL kernels.
- Fixed AppArmor enforcement when KubeArmor was deployed in namespaces other than `kubearmor`.
- Resolved multiple issues in systemd-based release CI pipelines.
